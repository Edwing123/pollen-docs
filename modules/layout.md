# Layout

### Size Scale

| Property group | Applies to |
| :--- | :--- |
| `--size-*` | Any size-related property \(`margin`, `padding`, `width`,`height`, etc\) |

section {
  margin-top: var(--size-5);

| Property group | Applies to |
| :--- | :--- |
| `--width-*` | `max-width` |
